MACOS

Su Mac non serve una versione diversa dell'app: devi copiare tutta la cartella `transferimento-cockpits`.

Per semplicita`, i file da aprire su macOS sono raccolti qui dentro:

- `avvia-transferimento-cockpits.command`: apre la GUI
- `export_fast.command`: crea lo ZIP di export
- `import_fast.command`: importa l'ultimo ZIP trovato
- `0-mac-fix-permissions-and-launch.command`: prova a togliere la quarantena macOS e poi avvia l'app

Questi launcher lavorano sulla cartella padre, quindi non spostare da soli i soli file `.command`: devono restare dentro la cartella completa dell'app.
